## Gameplay Instructions

1. **Installation**
   - Make sure you have Python 3.x installed on your machine.
   - Clone or download the game repository from [GitHub Repository Link](insert GitHub repository link here).
   - Navigate to the project directory in your terminal/console.

2. **Running the Game**
   - Run the following command to start the game:
     ```
     python main.py
     ```
   
3. **Gameplay**
   - Enter your name to start the game.
   - Follow the prompts and instructions displayed on the screen to progress through the game.
   - During battles, choose from the available options:
     - **1. Attack**: Perform a regular attack on the enemy.
     - **2. Use Health Potion**: Restore some health by using a Health Potion from your inventory.
     - **3. Super Attack**: Perform a powerful super attack on the enemy (available once per enemy encounter).
     - **4. Run**: Attempt to escape from the battle.
   - Level up your character by defeating enemies and gaining experience points.
   - Customize your character's attributes, abilities, and equipment as you level up.
   - Face tougher enemies and bosses as you progress in the game.
   - Make strategic choices that shape the storyline and lead to different outcomes.

4. **Winning and Losing**
   - Defeat the Mutilator boss at level 10 to win the game.
   - If your character's health reaches zero, you will be defeated, and the game will end.
